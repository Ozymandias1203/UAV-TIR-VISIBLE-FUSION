# dataset_profile.md

## 1. 文档目的

本文档记录本项目处理数据的不可变物理事实。它只描述数据集边界、影像分辨率、标定板几何、相机与元数据基线，不描述算法，也不描述运行时参数。

## 2. 数据边界

- 业务处理数据集：`test_Arctic/`
- `test_Arctic/tiff_dir/` 中的 TIFF 热红外帧是本项目热红外主处理对象。
- `test_Arctic/thermal_dir/` 中的 JPG 热红外帧仅保留为预览、副本和人工检查材料。
- 标定数据集：`M400-H30T-CALIB-CHESSBOARD/`
- 影像元数据：`metadata_all.json`
- 初始相机标定：`H30T_RGB.xml` 与 `H30T_NIR.xml`

## 3. 不可变物理事实

### 3.1 影像分辨率

- 可见光影像分辨率：4032 × 3024 px
- 热红外影像分辨率：1280 × 1024 px

### 3.2 标定板几何

- 标定模式采用 12 × 9 棋盘格识别目标

上述尺寸是后续棋盘格检测、角点筛选和质量门槛的依据。若未来标定板更换或重新测量，必须先更新本文件，再更新其他设计文档。

### 3.3 相机与元数据基线

- 可见光通道来自 H30T 可见光相机
- 热红外通道来自 H30T 热红外相机
- `H30T_RGB.xml` 和 `H30T_NIR.xml` 分别记录两路相机的初始几何参数
- `metadata_all.json` 记录 ExifTool 提取到的帧级元数据与可见光 / 热红外共同的设备信息

### 3.5 元数据观察值

从 `metadata_all.json` 当前样本中可稳定观察到以下字段或量级：

- `FocalLength` 约为 6.7 mm
- `FNumber` 约为 1.7
- `ISO` 约为 130
- `ExposureTime` 约为 1/1500 s
- `SensorTemperature` 约为 4.0 °C
- `LensTemperature` 约为 5.1–5.5 °C
- `LRFTargetDistance` 约为 199.4–200.0 m
- `LightValue` 约为 11.4–11.7 EV
- `RTK` / GPS 相关精度字段约为 0.07679 m 量级
- `ExifToolVersion` 已记录在元数据文件中，必须随处理结果保留

## 4. 语义约束

- `H30T_RGB.xml` / `H30T_NIR.xml` 中的 `f` 是标定模型中的像素焦距，不等于 ExifTool 的 `FocalLength` 毫米值。
- `cx` / `cy` 在本项目内部必须被转换为绝对像素主点坐标后再进入后续模块。
- `SensorTemperature` 和 `LensTemperature` 是设备诊断值，不是环境空气温度的替代品。
- `LRFTargetDistance` 是设备侧距离参考值，不是辐射校正唯一依据。

## 5. 同步纪律

- 任何更换相机、镜头、固件、标定板或影像尺寸的变化，都必须先更新本文件。
- 本文件一旦更新，必须同步更新 [AGENTS.md](../AGENTS.md) 与受影响的设计文档。
- 其他文档若需要引用物理尺寸、分辨率或基线参数，应直接链接本文件，不得重复写成另一套“事实表”。
