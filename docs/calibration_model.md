# calibration_model.md

## 1. 文档目的

本文档定义双光谱标定层的设计：如何从 H30T RGB 与热红外标定图像中估计相机几何参数，如何进行去畸变，以及如何评价标定结果是否可用于后续匹配、投影和热富集。

## 2. 适用范围

- 适用于 H30T RGB 相机与热红外相机的独立内参估计。
- 适用于 9×6 棋盘格标定数据。
- 适用于后续去畸变和单应性估计所需的几何基础图像生成。
- 不负责热辐射校正，也不负责三维重建。

## 3. 相关输入

### 3.1 图像输入

- 可见光标定图像：来自 RGB 传感器。
- 热红外标定图像：来自热红外传感器。
- 标定图像应尽量覆盖棋盘格的不同姿态、位置和成像范围。

### 3.2 参考输入

- H30T_RGB.xml
- H30T_NIR.xml
- 标定板影像数据
- 参考文献中的标定方法描述

### 3.3 先验信息

- 图像尺寸
- 棋盘格角点数量和排列
- 相机工作模式与时间戳
- 若可用，则使用设备初始标定参数作为初值

## 4. 前处理策略

### 4.1 RGB 图像

- 对比度归一化
- CLAHE 局部增强
- 必要时做轻微平滑以减少噪声对角点检测的影响

### 4.2 热红外图像

- 灰度反转，以便让棋盘格结构更接近可检测的亮暗分布
- Otsu 自动二值分割
- 局部热对比度增强
- 必要时做连通域清理，提升棋盘格边界稳定性

### 4.3 前处理约束

- 前处理只服务于角点检测和亚像素定位，不修改原始图像文件。
- 前处理参数必须可复现，并写入标定结果摘要。

## 5. 标定模型

### 5.1 参数集合

对于每个传感器，至少应保存以下参数：

- `image_width`
- `image_height`
- `focal_length`
- `principal_point_x`
- `principal_point_y`
- `affinity_or_skew`
- `radial_coefficients`
- `reprojection_rms`
- `quality_flag`

### 5.2 参数语义

- 主点坐标必须明确是绝对像素坐标还是相对图像中心的偏移量。
- 畸变参数必须与实际使用的模型一致，不得将不同模型的符号约定混用。
- 若某一版输出采用 H30T XML 风格的保存格式，应保持与输入字段的映射可追踪。

### 5.3 与 H30T XML 的对齐原则

- XML 中的宽高字段应与图像真实分辨率一致。
- XML 中的 `f`、`cx`、`cy`、`b1`、`k1`、`k2`、`k3` 应被视作当前设备链路的核心几何参数。
- 若输出到其他系统时需要转换为绝对主点坐标，转换规则必须在文件格式层显式记录。

## 6. 标定流程

### 6.1 角点检测

1. 读取预处理后的标定图像。
2. 检测棋盘格角点。
3. 对角点进行亚像素精化。
4. 对低质量图像或低置信角点进行剔除。

### 6.2 参数求解

1. 建立二维棋盘格角点与三维平面坐标的对应关系。
2. 使用初值或参考参数初始化内参。
3. 估计内参和畸变参数。
4. 计算重投影误差并生成质量标记。

### 6.3 迭代策略

- 若初次标定 RMS 偏高，应先检查角点检测质量，再考虑剔除异常图像或异常点。
- 若某一组图像明显偏离整体，应优先作为外点候选，而不是强行保留。
- 迭代目标不是无限逼近数值最小值，而是获得在后续几何任务中稳定可用的结果。

## 7. 去畸变设计

### 7.1 目标

将原始图像变换为更符合针孔模型的几何基础图像，为匹配和投影提供稳定输入。

### 7.2 规则

- 去畸变只能使用已经验证过的标定参数。
- 去畸变过程中必须保留映射关系、裁切信息和缩放信息。
- 去畸变图像不能被当作原始传感器数据替代品。

### 7.3 输出内容

- 几何校正后的图像
- 变换元数据
- 裁切或重采样记录
- 原图与去畸变图的索引关系

## 8. 质量门槛

### 8.1 基本门槛

- 棋盘格识别率达到项目要求。
- 重投影误差处于可接受范围。
- 主要参数在物理上合理，例如主点不会偏离图像中心过远，焦距不应出现异常值。

### 8.2 诊断门槛

- RGB 与热红外的标定质量应分别记录。
- 若某一传感器的质量明显低于另一传感器，应单独标记，不要用平均值掩盖问题。
- 标定结果必须能为后续匹配和热富集提供稳定几何基础。

## 9. 输出建议

建议每次标定至少输出以下三类结果：

- 机器可读参数文件
- 人工可检查的角点覆盖图
- 重投影误差统计摘要

## 10. 非目标

- 不是把热红外与可见光强行合并成同一相机模型。
- 不是为了追求最低 RMS 而忽略后续链路可用性。
- 不是辐射校正模块。
